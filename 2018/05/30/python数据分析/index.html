Python数据分析基本环境搭建（mac）

  • undefined
  • 2018-05-30

    在使用Python进行数据分析的过程中，我们一般会使用到的一些科学计算库有：Numpy，Scipy、matplotlib（用于生成图表），还有Ipython工具等。

    我们可以在自己的机器上一个个安装这些库，也可以直接安装EPDFree(Enthought Python Distribution),这是一个包含了所有这些科学计算库的包。直接安装可以避免很多环境及版本差异导致的问题。

    安装之前请确保已经安装路Xcode
    安装步骤：
    1、启动终端输入
    $gcc -v会出现”Configured with…”的信息，则表示Xcode安装成功
    2、安装Cnopy(以前叫EPDFree）,下载dmg文件，双击进行安装。
    安装完之后基本环境就ok了.

    现在Cnopy相对于EPDFree来说包含的包更完善了，安装完后连ipython都直接集成在里面了，可以直接进行数据分析，省掉了很多环境配置的麻烦。点击下图Editor就可以进入ipython环境

    至此，可以开始分析工作了
